## Getting Started

APP desarrollada con nextJs, donde se encuentran los comandos iniciales que son:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

[http://localhost:3000](http://localhost:3000)

## La APP

- Es un juego sencillo sobre buscar las imagenes identicas y realizar un match, se encuentra en etapas iniciales asi que existe una logica basica de como obtener las imagenes de pokemons.

- La app utiliza la API de pokemons para la obtenecion aleatoria de los primeros 151 pokemons o la primera generación, una vez obtenida las imagenes se renderiza el juego.
